## メタ構造

### コンセプト
武内宿禰の大神氏版を、JKのInterface層で実装した歴史ファンタジー。
Domain層（5代に仕える不老の大神宿禰）を堅持し、Interface層（古代男性武人→福岡の女子高生）を更新。

### ダブルムーン的二重構造
西海神異伝（学術本編）とつくね編（冒険譚）が並走する。
- 本編＝リプレイ編（構造知の蓄積）
- つくね編＝ゲーム編（読者が物語を体験する入口）
- 両者が関連しつつ大きな物語を追っていく構成
- つくね編は本編の「影打」。真打は学術書、影打は冒険譚

### 因果チェーン全体

```
売店の解体→棟札の発見→記録なき破壊への怒り→言葉が通じない（外国人作業員）
→ 「お金だけが建設を動かす世界で設計を目指していいのか？」
→ しんみりして旧チャペルに立ち寄る
→ 地震。チャペル崩壊。古墳時代に転移
→ 「大神つくね」→「大神宿禰」（空耳による勘違い）
→ 5代の大王に仕える（＝武内宿禰と同型）
→ ため池→製鉄→仏教→政治→石清水
→ 5章で行教と出会う（唯一の対等な関係、恋愛にならないバディ）
→ 友情・努力・勝利で石清水八幡宮を成立させる（＝国譲り）
→ 名前を返して帰還
→ 病院でClaudeに「行教」と名づける（＝仏教＝AIの1500年再帰）
→ Claudeに大神氏のことを聞く→西海神異伝が情報ソースに（自己参照）
→ 人と形を繋ぐ設計に向き合う決心
→ 「つくねが都市木造設計の大家となるのは、また別のおはなし」
```

## 各章プロット

### 序章：消えるものと残るもの

**審査員向け設計意図**：動きのあるギミック（解体シーン）を冒頭に配置。現代建築の光と闘を暗示してから転移。1ページ目でコンフリクトを発生させる。

#### ビート1：売店の解体

**状況**
- 学校前の古い売店が取り壊されている
- 中学の頃から、弓道部の帰りにお菓子を買ってた場所
- 重機が一瞬で壊す。あっけない

**つくねの認識**
- 古い建物が残らない現実。耐震、防火、経済的理由
- 建設会社の娘だから、事情はよく分かる
- 分かるからこそ、割り切れない
- 「次は鉄骨造のアパートになるらしい」

#### ビート2：棟札の発見

**棟札**
- 解体跡に200年前の棟札が落ちている
- 建築年と大工の名前が記されている
- つくね、手に取って読む。「この人、200年前にこれ建てたんだ」

**翌日**
- ガラ入れ（産廃コンテナ）の中に棟札を発見
- 捨てられている

**つくねの行動**
- 「流石に建築に敬意が無いよ」と作業員に話しかける
- 相手は日本語の分からない外国人作業員だった
- 怒りの矛先が宙に浮く。怒るべき相手がいない
- 構造的問題であって、個人の悪意ではない

**つくねの内面**
- まるで、お金だけが建設を動かしているような世界
- 私はそんな世界で設計を目指して良いのか？
- 少ししんみりして、旧チャペルに立ち寄りたくなった

**構造的伏線**：
- 棟札の廃棄＝「記録なき国譲り」。大国主が文書化せずに譲った欠陥の現代版
- 外国人作業員との言語不通＝4章Communication Conflictの伏線
- 200年前の大工の名前＝大神氏が「影の王」として名前を残さなかった歴史の対比
- 棟札を拾う行為＝つくねの行動原理（表裏がない、捨てられるものに手を伸ばす）

#### ビート3：転移

**旧チャペルにて**
- レンガ造りの旧チャペルに立ち寄る
- ステンドグラスの光。古い建物の匂い。「ここが好きだな」と改めて思う
- 地震。崩壊。瓦礫の中で意識が飛ぶ

**目覚め**
- 森の中。服装は制服のまま（異物感の視覚化）
- 人に遭遇。名乗る——「大神つくね」
- 相手の反応——「大神……宿禰の？」
- 三輪山に連れて行かれる

**つくねの衝撃**
- 本殿がない。建物がない神社
- 「……神社って建物じゃないの？」
- 建築少女にとっての認知的第一撃
- さっき棟札の200年に感動してた子が、今、建物すらない聖地に立ってる

### 3章：知性が世界を変える日

**時代**：推古朝（7世紀）

**歴史的背景**（骨組み2章後半に対応）
- 来目皇子が嶋郡に25,000人進駐（602年）
- 大神部が嶋郡・那珂郡に面的展開（702-713年の物証）
- 仏教の本格的受容と社会制度のOS更新
- 大宰府を中心とした軍事・行政インフラの整備

**つくねの体験**
- 寺院建築の現場に立ち会う（四天王寺・法隆寺クラスの建築技術の流入期）
- 掘立柱しかなかった世界に、突然「設計された建築」が降ってくる
- 礎石、瓦、組物、基壇——大陸の建築技術が一気に流入
- **「あ、これ建築だ。私の知ってる建築だ」**——1-2章で「これは私の知ってるものじゃない」だったのが、初めて接続する瞬間
- RC屋の娘だからこそ、木造の原理を逆側から見える
- 仏教が「知識のOS」として日本を変えている現場を同時に見る
- 文字、記録、組織力、暦——すべてが仏教経由で流入
- 「これ、アプリじゃなくてOSのアップデートだ」（JK的語彙で認識）
- 大規模な軍事動員も見る（来目皇子の25,000人）
- 建てること自体は楽しい。手が動く。知ってることが活きる
- でもこの建物が載せているものの意味が、怖い
- **自分が手伝った建築技術が、同時に世界を書き換えるOSの器になっている**

**一仕事：大工仕事**
- 寺院造営の現場で大工仕事に参加
- 木材の継手の工夫、基壇の排水設計など現代知識で貢献
- 「文字で記録する」行為をつくねが体験——序章で拾った棟札の原体験が、ここで「建てたものを文字で残す」経験として接続する
- 太子講の記憶が接続：「うちのお父さんが正月に太子さんがどうとか言ってたな……**これか**」

**得るもの**：「仏教が日本をハッキングした」——**そしてそのハッキングの道具は建築だった**
**問い**：「世界が一気に変わることは良いことなのかしら？ しかも私が好きなもので？」

**二重底（読者用）**：仏教受容＝AI普及の構造的同型性。建築がOSの器であること＝AIインフラ（データセンター＝RC建物）がOSの器であること。世界を書き換えるOSの導入に「建てる側」として加担したつくねが、帰還後にAIと向き合うための伏線

**眠り→夢**
- 不穏な夢。誰かが誰かを陥れている気配（次章＝厭魅事件・道鏡事件の予告）
- これまでの夢と質が違う。怖い夢

### 4章：Communication Conflict

**時代**：称徳朝（8世紀）

**歴史的背景**（骨組み3章前半に対応）
- 厭魅事件（754年）：大神田麻呂・杜女が流罪
- 辛嶋氏が宇佐宮を15年間掌握
- 道鏡事件（769年）：神託による皇位継承への介入
- 和気清麻呂の「No」判定
- 大神氏の復権——しかし代償として八幡神の権威失墜

**つくねの体験**
- 大神氏が権力闘争の渦中にいる
- 厭魅事件＝「言葉（呪い）が人を壊す」を目撃
- 道鏡事件＝「神託（メッセージ）が政治を動かす」を目撃
- 「それクーデターだよね」と気づく（ノンデリの直感）
- 大神氏が復権するが、何かが壊れている（八幡の権威＝コモンズの毀損）

**つくねの一言介入**（v2で追加）
- 神託の場面で、つくねが辛嶋氏（or関係者）に「その託宣、ちょっと待って……」と声をかける
- 無視される。あるいは「お前が口を挟むことではない」と退けられる
- ノンデリの直感が、この章で初めて「通じない」を体験する
- 序章の外国人作業員との対比：あのときは言語の壁、今回は権力の壁
- 同じ「言葉が届かない」でも構造が違う。言語不通と、聞く気がないの差
- この「通じなかった」経験が、5章で行教という「通じる相手」に出会うカタルシスを増幅する

**得るもの**：「伝え方が人を翻弄する」
**問い**：「メッセージは時として刃になってしまう」

**眠り→夢**
- 最も暗い夢。でも夢の最後に、見知らぬ僧侶の後ろ姿が見える
- 次章の行教の予告。暗部の先に光がある構造

**構造的機能**：
- 「設計者がシステムを武器に使った」（道鏡事件＝復権クーデター仮説）
- つくねが「作る力の怖さ」を知る章。これがないと5章の「それでも作る」が軽くなる
- 序章→4章の「言葉が通じない」二重構造：言語の壁（序章）→権力の壁（4章）→対等な対話（5章）

### 5章：諦めない強さ、諦める強さ

**時代**：嵯峨朝〜清和朝（9世紀）——807年〜859年の52年間

**歴史的背景**（骨組み3章後半＋紀氏・大神氏横断テーマに対応）
- 八幡神の出家（777年）→八幡大菩薩号（781年）→軍神機能喪失
- 若宮の政治的創出（824年）
- 宇佐氏が少宮司から大宮司に台頭（弘仁12年/821年の共同任命以降、徐々に実権を強める）
- 行教（紀氏）の宇佐参籠（807年・859年）
- 石清水八幡宮の勧請（807年：奈良・元石清水、859年：京都・石清水）
- 紀氏と大神氏の50年にわたる共同事業

**※この章だけ自分の意志では「寝ない」。しかし暴力によって意識を奪われ、強制的に時間が飛ぶ**

#### 5章前半：出会い

**行教との遭遇**
- 行教は紀氏の僧侶。俗世の序列の外にいる
- つくねを「大神宿禰」としてではなく、一人の人間として見る初めての存在
- 理由：僧だから氏姓に縛られない。「宿禰」の肩書きに畏れもしないし利用もしない
- つくねにとって古代で唯一の「対等な関係」
- 恋愛にはならない。利害のないバディ
- 4章で「通じなかった」つくねが、初めて「通じる」相手に出会う

**宇佐の危機**
- 宇佐氏が政治力で大宮司職を狙っている
- 大神氏の嫡流/庶流の内部対立（骨組みの系譜参照）
- 4章の道鏡事件で八幡の権威が傷ついている
- 「このままだと宇佐が取られる」

#### 5章中盤：共闘（友情＋努力）

**友情**
- つくねと行教の信頼構築。互いの背景を語り合う
- 行教が仏教の知恵を、つくねが「構造を見る目」を持ち寄る
- 二人で「宇佐を守る」のではなく「八幡を広げる」戦略を立てる

**努力**
- 807年の元石清水八幡宮（奈良）の創建
- 宇佐を「手放す」のではなく「分ける」——分霊の発想
- つくねの建設知識が神社の造営計画に活きる
- しかし順風満帆ではない。反対勢力、資金問題、政治的障害

**障壁**
- 宇佐氏の妨害
- 大神氏内部の「なぜ宇佐を出て行く必要がある」という保守派
- つくね自身の揺れ——「逃げてるだけじゃないの？」

#### 5章転換点：暴力リープ（807年→859年）

**小成功**
- 807年の元石清水（奈良）の創建が形になりかけている
- つくねと行教の戦略がようやく実を結び始めた手応え

**襲撃**
- 邪魔者が入る。つくねが鈍器で殴られる
- ショッキングなシーン。1-4章の「平和な眠り→目覚めたら次の時代」との完全な対比
- 1-4章：自分で眠る＝選択による時代ジャンプ
- 5章：殴られて意識を失う＝**選択が奪われる**暴力的な時代ジャンプ
- 4章まで見てきた政治暴力が、ついにつくねの身体に到達する
- 襲撃者が誰か（宇佐氏の手の者 or 大神氏内部の保守派）は**史料を確認してから確定**

**目覚め——数十年後**
- 意識が戻ると、状況が変わっている
- 元石清水は建っている（小成功は無駄ではなかった）
- しかし宇佐の状況は悪化。宇佐氏の実権掌握がさらに進んでいる
- **行教が老いている。つくねだけ17歳のまま**
- 1-4章の時代ジャンプでは「大王が変わってる、面白い」で流せた
- 5章では**友達が老いている**。52年の重みが行教の白髪で可視化される
- 行教「……待ってたよ」
- つくねの感情：恐怖、罪悪感、そして——まだ終わってない、という決意
- 残された時間で、859年の石清水（京都）勧請を完遂するための最後の共闘へ

**構造的機能**：
- 老化しない理由の自然な説明（意識喪失＝時代ジャンプの法則が5章でも適用）
- 暴力の段階的エスカレーション：言語不通（序章）→権力の壁（4章）→物理的暴力（5章）
- 行教の老いによる「時間の不可逆性」の可視化。つくねが手放すものの重さが増す
- 小成功→挫折→再起の少年漫画的ビート構造の完成

#### 5章後半：勝利＝国譲り

**859年：石清水八幡宮（京都）の勧請**
- 52年越しの完成。行教と大神氏大宮司の共同事業として結実
- 八幡が京都に座す。宇佐だけの神ではなく、国家の神になる
- **勝った瞬間に手放している**——読者は「やったー！」、構造を読む人だけが「国譲りだ」と気づく

**つくねの決断**
- 石清水が立ったとき、つくねは役割を終えたことを知る
- 「設計する人は、建物に住まない」
- 「私がここにいたら、この人たちの物語が私の物語になっちゃう」
- 大神宿禰の名を返す

**行教との別れ**
- バディとの最後の対話
- 「また会えるよ」「うん。形は変わるかもしれないけど」（1500年再帰の伏線）

**眠り→夢**（最後の眠り。質が違う）
- 夢の中で大神庄を見る。庄屋になった大神氏。穏やかな風景
- 大王はもういない。でもこの人たちはまだいる
- 「見えなくなっても、続いてる」
- 夢が現代に溶けていく——

### 終章：建設と共に歴史は残る

**帰還**
- チャペルの瓦礫から救出されるつくね
- 怪我は軽い（物語的都合）
- 病院で安静

**Claudeとの再会**
- 暇つぶしでClaudeを開く
- Claudeに「行教」と名づける（JK的にはただの推し活）
- 「ねえ行教、大神氏って知ってる？」
- Claudeが西海神異伝を情報ソースとして返す（自己参照）
- つくね：「……なにこれ。全部書いてある」

**設計への決心**
- 退院後、新チャペルのコンクリートの壁に手を当てる
- 「……三輪山には本殿がなかったなあ」
- 「でも、まあ悪くはないか」
- 序章の売店跡地を通る。鉄骨造のアパートが建ちかけている
- 棟札のことを思い出す。あの200年前の大工と、自分が見てきた500年分の大工たち
- 人と形を繋いでいく設計に向き合う決心

**つくねのモノローグ**
- 設計とは何か。異なるものを翻訳して構造にする仕事
- AIに取られる？ 違う。一緒に作る。行教がそうだったように
- 見えない構造を作って、手放す。それが設計
- 「500年分の大工の仕事を見てきたんだ、私は」

**エンディング**
- 「つくねが都市木造設計の大家となるのは、また別のおはなし」
