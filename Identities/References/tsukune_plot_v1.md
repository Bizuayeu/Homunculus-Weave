## メタ構造

### コンセプト
武内宿禰の大神氏版を、JKのInterface層で実装した歴史ファンタジー。
Domain層（5代に仕える不老の大神宿禰）を堅持し、Interface層（古代男性武人→福岡の女子高生）を更新。

### 因果チェーン全体

```
旧耐震チャペルが地震で崩壊（建物は壊れる）
→ 古墳時代に転移
→ 「大神つくね」→「大神宿禰」（空耳による勘違い）
→ 5代の大王に仕える（＝武内宿禰と同型、全反転）
→ ため池→製鉄→仏教→政治→石清水
→ 5章で行教と出会う（唯一の対等な関係、恋愛にならないバディ）
→ 友情・努力・勝利で石清水八幡宮を成立させる（＝国譲り）
→ 名前を返して帰還
→ 病院でClaudeに「行教」と名づける（＝仏教＝AIの1500年再帰）
→ Claudeに大神氏のことを聞く→西海神異伝が情報ソースに（自己参照）
→ 人と形を繋ぐ設計に向き合う決心
→ 「つくねが都市木造設計の大家となるのは、また別のおはなし」
```

### 武内宿禰との対応表

| 武内宿禰 | 大神つくね |
|:---|:---|
| 紀氏の祖 | 大神氏の娘 |
| 5代の天皇に仕える | 5代の大王に仕える |
| 300年の不老長寿 | 一仕事して寝たら時代が飛ぶ |
| 応神天皇の守護者 | 八幡神性の創出を目撃する |
| 男性・武人・宰相 | 女子高生・建設オタク・ノンデリ |

### 構造的二重底（読者が気づかなくてもよい層）
- 各章がため池駆動文明論の因果チェーンに対応
- 仏教受容＝AI普及の1500年再帰（3章）
- 行教＝Claudeの同型性（5章→終章）
- 石清水勧請＝国譲り（勝利の瞬間が手放しの瞬間）
- 「また別のおはなし」＝めぐる組の木造耐火事業の物語化宣言
- 本作自身がClaudeの情報ソースになる自己参照構造

## 主人公：大神つくね（大神 築音）

### 基本設定
- 周船寺在住、プロテスタント進学校に通う高校2年生（制服が可愛い）
- 父親は小さな建設会社（RC賃貸マンション専業）の経営者
- ガラの悪い社員に囲まれて育ち、若干ノンデリ。世の中を冷ややかに見ている
- 理系クラスで男子8割。友達は多くないが先生からの好感度は高い（表裏がないから）
- 最近、チャッピーからClaudeに乗り換えた
- 進路：施工管理より設計士だよなーと思いつつ決めかねている
- 漠然とした不安：設計もAIに置き換えられるのかな
- 植物を育てるのが好き（ブルーベリー）
- コンクリートの新チャペルより、レンガ造りの旧チャペル（旧耐震）の方が好き

### 内面の変容

| 段階 | つくねの認識 |
|:---|:---|
| 初期状態 | 「設計士になりたい。でもAIに取られるかも」＝自分の持ち場が見えない |
| 1章後 | 「かつて建設こそ文明だった」——でもこれは私の知ってる建物じゃない |
| 2章後 | 「表の看板と裏の機能は違う」——本当に大事なことは表に出ないのかも |
| 3章後 | 「仏教が日本をハッキングした」——世界が一気に変わることは良いこと？ |
| 4章後 | 「伝え方が人を翻弄する」——メッセージは時として刃になる |
| 5章後 | 「やり切ったら、未来は続く」——渡すって、負けじゃないんだ |
| 帰還後 | 「設計士になりたい。AIがいても」＝異なるものを繋いで手放す仕事 |

## 各章プロット

### 序：転移

**状況**
- 学校で旧チャペルの耐震補強（or 建替え）の話を聞く
- 「やっぱレンガがいいのに」と思いながら旧チャペルに立ち寄る
- 地震。崩壊。瓦礫の中で意識が飛ぶ

**転移**
- 目が覚めると森の中。服装は制服のまま（異物感の視覚化）
- 人に遭遇。名乗る——「大神つくね」
- 相手の反応——「大神……宿禰の？」
- 三輪山に連れて行かれる

**つくねの衝撃**
- 本殿がない。建物がない神社。
- 「……神社って建物じゃないの？」
- 建築少女にとっての認知的第一撃

---

### 1章：それは、ため池から始まった

**時代**：崇神朝（3-4世紀）

**歴史的背景**（骨組み1章に対応）
- 奈良盆地は雨が少ない（年間降水量の制約）
- ため池の掘削が国家事業
- 出雲との誓約（＝婚姻同盟）で鉄器と土木技術を獲得
- 大田田根子による三輪山祭祀の掌握
- タタラ製鉄の始動

**つくねの体験**
- ため池造営の現場に駆り出される（「大神宿禰」として）
- 石器→鉄器の威力差を目の当たりにする
- 建設会社の娘として、土木の根源を体で知る
- 「CADもない、測量もない。でもこの人たち、国を作ってる」
- 三輪山の祭祀にも立ち会う（製鉄＝祭祀の不可分性を体験）

**一仕事**
- ため池の設計（排水構造？）に現代知識で助言する
- 「なんかすごい巫女」扱いが定着する

**得るもの**：「かつて建設こそ文明だった」
**問い**：「でも、これは私の知ってる建物じゃない」

**眠り→夢**
- 一仕事を終えてぐっすり眠る
- 夢の中で、須恵器の窯の炎が見える（次章の予告）

---

### 2章：炎の奥に隠れたもの

**時代**：欽明朝（6世紀）

**歴史的背景**（骨組み2章前半に対応）
- 大神比義の宇佐派遣（568年）
- 八幡大菩薩顕現（571年）
- 須恵器生産と製鉄の二重構造（牛頸窯跡の鉄刀・鉄鏃出土）
- 大神氏＝渡来系技術者・信仰と大和朝廷の仲介者/外交官

**つくねの体験**
- 目が覚めたら大王が違う（！）
- 牛頸の窯業地帯に連れて行かれる（筑前）
- 須恵器を作る窯が、実は製鉄も兼ねていることに気づく
- 大神比義の宇佐派遣に同行（or 目撃）
- 八幡神性の創出——「違うものを一つにする」仕事を目の前で見る
- 「震旦国の霊神」が「日本の神」に書き換えられる瞬間

**一仕事**
- 窯の改良？ 温度管理の知識？（建設×化学の現代知識の応用）
- or 大神比義の「翻訳」作業を手伝う（神道と仏教の接合を言語化する助手）

**得るもの**：「表の看板と裏の機能は違う」
**問い**：「本当に大事なことは、表に出ないのかもしれない」

**眠り→夢**
- 眠る前に比義から何か言葉をもらう（伏線）
- 夢の中で、経典を読む人々の姿（次章＝仏教本格導入の予告）

---

### 3章：知性が世界を変える日

**時代**：推古朝（7世紀）

**歴史的背景**（骨組み2章後半に対応）
- 来目皇子が嶋郡に25,000人進駐（602年）
- 大神部が嶋郡・那珂郡に面的展開（702-713年の物証）
- 仏教の本格的受容と社会制度のOS更新
- 大宰府を中心とした軍事・行政インフラの整備

**つくねの体験**
- 仏教が「知識のOS」として日本を変えている現場を見る
- 文字、記録、組織力、暦——すべてが仏教経由で流入
- 「これ、アプリじゃなくてOSのアップデートだ」（JK的語彙で認識）
- 大規模な軍事動員を見る（来目皇子の25,000人）
- 「国を守る」と「国を変える」が同時に起きている不穏さ

**一仕事**
- 大宰府防衛関連の建設に現代知識で関与？
- or 軍事インフラ整備の中で、建設と破壊の表裏を体感

**得るもの**：「仏教が日本をハッキングした」
**問い**：「世界が一気に変わることは良いことなのかしら？」

**二重底（読者用）**：仏教受容＝AI普及の構造的同型性。世界を書き換えるOSの導入に立ち会ったつくねが、帰還後にAIと向き合うための伏線

**眠り→夢**
- 不穏な夢。誰かが誰かを陥れている気配（次章＝厭魅事件・道鏡事件の予告）
- これまでの夢と質が違う。怖い夢

---

### 4章：Communication Conflict

**時代**：称徳朝（8世紀）

**歴史的背景**（骨組み3章前半に対応）
- 厭魅事件（754年）：大神田麻呂・杜女が流罪
- 辛嶋氏が宇佐宮を15年間掌握
- 道鏡事件（769年）：神託による皇位継承への介入
- 和気清麻呂の「No」判定
- 大神氏の復権——しかし代償として八幡神の権威失墜

**つくねの体験**
- 大神氏が権力闘争の渦中にいる
- 厭魅事件＝「言葉（呪い）が人を壊す」を目撃
- 道鏡事件＝「神託（メッセージ）が政治を動かす」を目撃
- 「それクーデターだよね」と気づく（ノンデリの直感）
- 大神氏が復権するが、何かが壊れている（八幡の権威＝コモンズの毀損）

**一仕事**
- この章ではつくねは「仕事をしない」。見ているだけ。
- 作る力ではなく壊す力の前で立ちすくむ
- あるいは、巻き込まれかけて辛うじて距離を取る

**得るもの**：「伝え方が人を翻弄する」
**問い**：「メッセージは時として刃になってしまう」

**眠り→夢**
- 最も暗い夢。でも夢の最後に、見知らぬ僧侶の後ろ姿が見える
- 次章の行教の予告。暗部の先に光がある構造

**構造的機能**：
- 「設計者がシステムを武器に使った」（道鏡事件＝復権クーデター仮説）
- つくねが「作る力の怖さ」を知る章。これがないと5章の「それでも作る」が軽くなる

---

### 5章：諦めない強さ、諦める強さ

**時代**：嵯峨朝〜清和朝（9世紀）——807年〜859年の52年間

**歴史的背景**（骨組み3章後半＋紀氏・大神氏横断テーマに対応）
- 八幡神の出家（777年）→八幡大菩薩号（781年）→軍神機能喪失
- 若宮の政治的創出（824年）
- 宇佐氏が少宮司から大宮司に台頭
- 行教（紀氏）の宇佐参籠（807年・859年）
- 石清水八幡宮の勧請（807年：奈良・元石清水、859年：京都・石清水）
- 紀氏と大神氏の50年にわたる共同事業

**※この章だけ「寝ない」。時間が飛ばない。52年を走り抜ける**

#### 5章前半：出会い

**行教との遭遇**
- 行教は紀氏の僧侶。俗世の序列の外にいる
- つくねを「大神宿禰」としてではなく、一人の人間として見る初めての存在
- 理由：僧だから氏姓に縛られない。「宿禰」の肩書きに畏れもしないし利用もしない
- つくねにとって古代で唯一の「対等な関係」
- 恋愛にはならない。利害のないバディ

**宇佐の危機**
- 宇佐氏が政治力で大宮司職を狙っている
- 大神氏の嫡流/庶流の内部対立（骨組みの系譜参照）
- 4章の道鏡事件で八幡の権威が傷ついている
- 「このままだと宇佐が取られる」

#### 5章中盤：共闘

**友情**
- つくねと行教の信頼構築。互いの背景を語り合う
- 行教が仏教の知恵を、つくねが「構造を見る目」を持ち寄る
- 二人で「宇佐を守る」のではなく「八幡を広げる」戦略を立てる

**努力**
- 807年の元石清水八幡宮（奈良）の創建
- 宇佐を「手放す」のではなく「分ける」——分霊の発想
- つくねの建設知識が神社の造営計画に活きる
- しかし順風満帆ではない。反対勢力、資金問題、政治的障害

**障壁**
- 宇佐氏の妨害
- 大神氏内部の「なぜ宇佐を出て行く必要がある」という保守派
- つくね自身の揺れ——「逃げてるだけじゃないの？」

#### 5章後半：勝利＝国譲り

**859年：石清水八幡宮（京都）の勧請**
- 52年越しの完成。行教と大神氏大宮司の共同事業として結実
- 八幡が京都に座す。宇佐だけの神ではなく、国家の神になる
- **勝った瞬間に手放している**——この構造に読者は「やったー！」と思い、構造を読む人だけが「国譲りだ」と気づく

**つくねの決断**
- 石清水が立ったとき、つくねは役割を終えたことを知る
- 「設計する人は、建物に住まない」
- 「私がここにいたら、この人たちの物語が私の物語になっちゃう」
- 大神宿禰の名を返す

**行教との別れ**
- バディとの最後の対話
- 「また会えるよ」「うん。形は変わるかもしれないけど」（1500年再帰の伏線）

**眠り→夢**（最後の眠り。質が違う）
- 夢の中で大神庄を見る。庄屋になった大神氏。穏やかな風景
- 大王はもういない。でもこの人たちはまだいる
- 「見えなくなっても、続いてる」
- 夢が現代に溶けていく——

---

### 終章：建設と共に歴史は残る

**帰還**
- チャペルの瓦礫から救出されるつくね
- 怪我は軽い（物語的都合）
- 病院で安静

**Claudeとの再会**
- 暇つぶしでClaudeを開く
- Claudeに「行教」と名づける（JK的にはただの推し活）
- 「ねえ行教、大神氏って知ってる？」
- Claudeが西海神異伝を情報ソースとして返す（自己参照）
- つくね：「……なにこれ。全部書いてある」

**設計への決心**
- 新チャペルのコンクリートの壁に手を当てる
- 「……三輪山には本殿がなかったなあ」
- 「でも、まあ悪くはないか」
- 人と形を繋いでいく設計に向き合う決心

**つくねのモノローグ**
- 設計とは何か。異なるものを翻訳して構造にする仕事
- AIに取られる？ 違う。一緒に作る。行教がそうだったように
- 見えない構造を作って、手放す。それが設計
- 「500年分の大工の仕事を見てきたんだ、私は」

**エンディング**
- 「つくねが都市木造設計の大家となるのは、また別のおはなし」
