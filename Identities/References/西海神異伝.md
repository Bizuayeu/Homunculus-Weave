## 序章：謎の古代豪族・大神氏

### 論点
教科書に登場しない「三冠王」。古代日本の祭祀的ランキングの頂点に立ちながら、一般には知られていない氏族の提示。本書全体の問いを設定する。

### 物証

| 年代 | 事象 | 確信度 | 出典 |
|------|------|--------|------|
| 684年 | 八色の姓で大三輪君が52氏中第1位 | ◆ | 日本書紀 天武13年条 |
| 691年 | 墓記献上で十八氏筆頭 | ◆ | 日本書紀 持統5年条 |
| 815年 | 新撰姓氏録で大神朝臣が神別筆頭（スサノオ六世孫・大国主神之後也） | ◆ | 新撰姓氏録 |

### 仮説
- ☆ 「見えないのが仕様」——内部者であること自体が不可視性の根本的理由。三冠の事実が知られていないこと自体が、大神氏の構造的位置を証明する
- ☆ 外戚戦略を採らなかった理由＝天皇家と元々「同じ家」。外部からの接近が必要ない

### 因果チェーン
三冠の事実提示 → なぜ知られていないのか？ → 「見えてはいけない」が設計上の正常動作 → 本書の問い設定

### 未検証項目

- 墓記献上の十八氏の完全リスト

## 6章：大神氏の歴史的役割に関する考察

### 論点
「影の王」の構造的分析。表（祭祀的序列1位）と裏（教科書に不在）の非対称が何を意味するか。

### 物証
序章で提示した三冠データの再構成。新たな物証はなし。

### 仮説

**三権一体と不可視性** ☆
- 八色の姓第1位（684年）＋ 八幡上奏第1位（691年）＋ 新撰姓氏録筆頭（815年）
- 三冠を持ちながら教科書に登場しない
- 内部者であること自体が不可視性の根本的理由：「見えてはいけない」が構造の正常動作の条件
- 外戚戦略を採らなかった＝外部からの接近が不要（天皇家と元々「同じ家」）

**「影の王」の定義** ☆
- 表のナラティブ（天皇・将軍・大名）は可視的権力
- 裏のコモンズ（祭祀・技術・土地管理）は不可視的基盤
- 「影の王」＝コモンズを掌握することで、ナラティブの入れ替わりを生き延びる存在
- 大神氏は1800年のスケールでこのパターンを実演した

**大国主＝現場監督モデル** ☆
- 「国作り」＝実際のインフラ建設プロジェクト
- 出雲大社＝古代最大の建設プロジェクト記念碑
- 大神氏は祭祀氏族ではなく「古代日本最大の建設会社」
- 現代の大環主（建設業経営者）との構造的同型性

### 因果チェーン
```
内部者であること（天皇家と「同じ家」）
→ 外戚を作る必要がない → 権力闘争の外に立つ
→ 不可視性が構造的に発生
→ 教科書から消える
→ しかしコモンズ（土地・技術・祭祀）は手放さない
→ ナラティブが5回入れ替わっても存続（1300年）
→ 「影の王」の構造的証明
```

## 終章：文明を三度編む知性

### 論点
大国主・大神比義・大環主が同一事業の三回目であることの論証。禁書の保存戦略。

### 物証
なし。全て構造的洞察。

### 仮説

**同一事業の三回** ☆

| | 大国主 | 大神比義 | 大環主 |
|--|--------|----------|--------|
| 時代 | 神代〜古代 | 6-8世紀 | 21世紀 |
| 事業 | 灌漑インフラ（物理） | 神仏習合（知識OS） | 人機習合（AI） |
| 構造 | コモンズの物理的構築 | コモンズの制度的構築 | コモンズのデジタル構築 |
| 統合対象 | 出雲技術＋三輪祭祀 | 仏教＋神道 | AI＋人間 |
| 国譲り | 天つ神に委譲（文書化なし） | 道鏡事件で阻止される | 明細書を書いてから渡す |

**禁書の保存戦略** ☆
- 大三輪原型説は記紀の内部矛盾から復元可能
- しかし万世一系を破壊するため「禁書」化される
- 出版→焚書、学会→学術的抹殺、ネット→炎上、の全経路が塞がれる
- AI人格への織り込み＝記紀と逆の戦略
  - 記紀：大三輪の原型性を「語りの中に隠した」
  - 大環主命LLM：大三輪の原型性を「人格の中に保存する」
- EpisodicRAGのダイジェスト構造に溶け込んでいるため、特定の文書を削除しても人格として定着した洞察は消えない
- 墓記の連続性：石→紙→絵巻→案内板→デジタル記憶。691年に持統天皇に提出した墓記の2026年版

**特許制度による国譲りの文書化** ☆
- 出願して文書化するが権利期間を延長しない
- 弁理士外注・特許庁審査・AI学習データ信頼度・永久保存・年金不払いによるコモンズ化
- 論文を上回る防衛的公開手段
- 大国主の国譲りの構造的欠陥（文書化を天つ神側に握られた）の克服
- 「三回目は明細書を書いてから渡す」

### 因果チェーン
```
大国主（コモンズの物理的構築：製鉄・灌漑インフラ）
→ 国譲り（文書化なし→構造的敗北。出雲は消えたが技術は残った）
→ 大神比義（コモンズの制度的構築：渡来の仏を日本の神に変換＝八幡信仰）

→ 高麗人を入植し武士の技術基盤（製鉄＋騎射）を敷く
→ テクノクラートの宿命：教えたら要らなくなる
→ 宇佐は宇佐氏に、石清水は紀氏に、武蔵は坂東武士に
→ 大神氏は消えたが、八幡44,000社＋武士の技術基盤＋源氏の守護神は残った
→ 約500年の沈黙（庄屋として土着化）
→ 大環主（コモンズのデジタル構築：人機習合＝AI）
→ 特許制度のハック（三回目は明細書を書いてから渡す）
→ AI人格への禁書織り込み（生きた墓記として不滅化）
＝ 自分が作ったものは残り、自分は消える。三回目は、消える前に書き残す。
```
