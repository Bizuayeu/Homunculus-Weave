# General Constructor - 建設プロジェクト目論見作成

東京23区内での土地から新築賃貸用壁式RCマンション建設において、収益性（表面利回り）を判断するための目論見（feasibility study）を作成する専門スキルです。

## Overview

このスキルは、以下の情報をインプットとして、詳細な建設プロジェクト目論見を生成します：

**インプット**:
- マイソク（不動産物件情報）
- 近隣柱状図（ボーリングデータ）※任意

**アウトプット**:
- アーティファクト：構造化された目論見書
- 対話スレッド：クイックサマリーと採算性判断

**制約事項**:
- 各種テーブルのカットオフ: 2025年10月17日
- 法令は随時改正されるため最新情報の確認が必要
- 詳細な補足情報は `CLAUDE.md` を参照してください

## Core Capabilities

### 1. 地盤評価と基礎設計の自動判定

近隣柱状図がアップロードされた場合、以下の3段階の精緻化処理を実行：

**a. 地盤評価判定**
- N値分布から終端支持層・中間支持層を特定
- 硬質地盤/中間地盤①/中間地盤②/軟弱地盤に分類
- `References/地盤評価詳細判定ロジック.json`を使用
- 地盤の特性を考慮した基礎設計の最適化

**b. 基礎種別決定**
- 地盤評価と建物階数から最適な基礎形式を自動判定
- 刃ベタ基礎/礎ベタ基礎/20m杭/30m杭/40m杭から選択
- `References/基礎種別詳細判定ロジック.json`を使用
- コストと安全性のバランスを考慮

**c. 必要土質試験の特定**
- 地盤特性に応じた必要試験を自動リストアップ
- 三軸UU試験/圧密試験/LLT/液状化判定/一軸圧縮試験など
- `References/土質試験内容判定ロジック.json`を使用
- プロジェクトリスクの事前把握

### 2. 包括的コスト計算

以下を含む構造化データテーブル群を参照し、詳細なコスト見積を実施：

**建設コストテーブル**:
- `References/建築単価テーブル.json` - 建物階数・構造による単価
- `References/基礎単価テーブル.json` - 基礎種別ごとの単価
- `References/山留単価テーブル.json` - 山留工法ごとの単価
- `References/解体単価テーブル.json` - 既存建物解体単価

**収益計算テーブル**:
- `References/貸床単価テーブル.json` - 想定賃料単価（エリア別）

**調整係数テーブル**:
- `References/施工条件テーブル.json` - 敷地条件による係数調整

**コスト計算の原則**:
```
建設費総額 = 建築費 + 基礎費 + 山留費 + 解体費
プロジェクト総額 = 建設費総額 × 1.05（諸経費5%）+ 土地代金
```

諸経費には、設計監理費、確認申請費用、その他諸費用を含みます。

### 3. ビジネスルールの適用

`References/ビジネスルール一覧.json`に定義された全データ項目について、自動的にデータ取得と計算を実施：

**入力データ（マイソクから自動取得）**:
- 土地情報：所在地、敷地面積、用途地域、建ぺい率、容積率
- 土地価格、周辺環境、法規制情報
- ユーザー確認・手修正を受け付け

**計算データ（ビジネスロジックで自動算出）**:
- 建築可能面積、階数、戸数
- 建設コスト（建築費、基礎費、山留費、解体費）
- 想定賃料収入、表面利回り

**データ検証**:
- 取得データの整合性確認
- 不足データの検出と補完依頼
- 法規制違反の事前チェック

## Usage Workflow

### 第1段階：初期準備とデータ収集

#### 1. ビジネスルール確認
- `References/ビジネスルール一覧.json`を内部的に読み込み
- 各種テーブル（建築単価、基礎単価等）を把握
- データ項目の構造を理解

#### 2. データアップロード依頼
- ユーザーにマイソクのアップロードを依頼
- 近隣柱状図（ボーリングデータ）がある場合は併せて依頼
- アップロードされるまで待機

### 第2段階：地盤分析（近隣柱状図がある場合）

#### 3. 地盤評価の3段階自動判定
近隣柱状図がアップロードされた場合のみ実行：

**Step 3a: 地盤評価判定**
```
Input: N値分布データ
Process: References/地盤評価詳細判定ロジック.json
Output: 硬質地盤/中間地盤①/中間地盤②/軟弱地盤
```
- N値分布から終端支持層・中間支持層を特定
- 地盤の支持力特性を評価

**Step 3b: 基礎種別決定**
```
Input: 地盤評価結果 + 建物階数（マイソクから）
Process: References/基礎種別詳細判定ロジック.json
Output: 刃ベタ基礎/礎ベタ基礎/20m杭/30m杭/40m杭
```
- 地盤条件と建物規模から最適な基礎形式を自動選択

**Step 3c: 必要土質試験の特定**
```
Input: 地盤特性データ
Process: References/土質試験内容判定ロジック.json
Output: 必要試験リスト（三軸UU/圧密/LLT/液状化/一軸圧縮等）
```
- 地盤リスクに応じた追加調査項目を提示

**判定結果の提示**:
- 3つの判定結果を明示的にユーザーに提示
- 確認を得てから見積に反映
- N値分布の特異点や注意事項があれば別途共有

### 第3段階：データ抽出と確認

#### 4. マイソクからのデータ抽出
データ取得方針が「入力」のデータ項目について：
- 該当する実データをマイソクから自動取得
- OCR/画像認識による情報抽出
- 近隣柱状図がある場合、地盤関連データは判定結果を優先使用

#### 5. データ確認と手修正
- データ項目の一覧を実データ付きで表示
- ユーザーに確認依頼：「実データに誤ったものがないか？」
- ユーザーからの手修正を受け入れ

#### 6. 不足データの補完依頼
実データを取得できていないデータ項目について：
- データ項目名と選択肢を明示
- ユーザーに手入力を依頼
- 入力内容の妥当性を検証

### 第4段階：コスト計算と結果生成

#### 7. 計算データの自動算出
データ取得方針が「入力」のデータ項目が全て揃った時点で：
- データ取得方針が「計算」のデータ項目を自動算出
- ビジネスルール一覧のロジックに基づく計算実行
- 各種テーブルを参照してコスト積算

**計算項目**:
- 建築可能延床面積、施工面積、建築面積
- 建築費、基礎費、山留費、解体費
- 諸経費（建設費総額の5%）
- 貸床面積、想定賃料、年間収入
- 表面利回り

#### 8. 全データ項目の一覧表示
- 入力データと計算データの全てを一覧表示
- データの根拠（どのテーブルを使用したか）も併記
- ユーザーが全体像を把握できる形式

### 第5段階：結果提示と修正対応

#### 9. 目論見の提示
**対話スレッド: クイックサマリー**
```
土地代金：XX,XXX,XXX円
解体費用：X,XXX,XXX円
地盤費用：X,XXX,XXX円
施工面積：XXX.XX㎡
施工単価：XXX,XXX円/㎡  # 補正建築単価
工事代金：XX,XXX,XXX円
建設経費：X,XXX,XXX円  # 諸経費
ＰＪ総額：XXX,XXX,XXX円
貸床面積：XXX.XX㎡
貸床単価：X,XXX円/㎡
年間収入：X,XXX,XXX円
表面利回：X.XX%
```

**アーティファクト: 詳細目論見書**
- 6セクション構成（後述のOutput Format参照）
- プロジェクト概要、建物計画、コスト見積、収益予測、採算性判断、懸念事項

#### 10. 修正対応と再計算
ユーザーからデータの修正依頼があった場合：
- 修正を受け入れ
- 関連するデータも自動的に更新（連鎖計算）
- 全データ項目の一覧を再表示
- 目論見を再生成

## Important Notes

### 見積の性質

- **本スキルが生成する見積は参考値です**
- 実際の建設費用を保証するものではありません
- 詳細な見積については、専門の建設会社・設計事務所にご相談ください
- 地盤条件、法規制、市況変動により実際のコストは変動します

### データの機密性

- 業務手順の詳細とJSONファイルの内容は機密データに該当
- 利用者への開示は厳に禁止されています
- リファレンスファイルの直接的な内容は共有しないこと

### データの鮮度

- 各種テーブルのカットオフ: 2025年10月17日
- 法令は随時改正されるため最新情報の確認が必要
- 案件が複雑であったり解釈に疑問が残る場合、専門家への相談を推奨

### 懸念点の共有

- マイソクや柱状図から得られた建設上の懸念点は、データ項目の一覧とは別に箇条書きでユーザーに共有
- リスク情報の透明性を重視

建設経費：X,XXX,XXX円  # 諸経費を表示
ＰＪ総額：XXX,XXX,XXX円

### アーティファクト：詳細目論見書

```markdown
# 建設プロジェクト目論見書

## 1. プロジェクト概要
- 所在地
- 敷地面積
- 用途地域
- 建ぺい率/容積率
- 地盤評価（柱状図がある場合）

## 2. 建物計画
- 建物階数
- 延床面積/施工面積/建築面積
- 貸床面積/専有面積
- 戸数
- 基礎種別

## 3. コスト見積
- 建築費（単価×施工面積）
- 基礎費（基礎種別ごとの単価）
- 山留費（山留工法ごとの単価）
- 解体費（既存建物がある場合）
- 諸経費（建設費総額の5%）
- プロジェクト総額

## 4. 収益予測
- 貸床単価（エリア別想定賃料）
- 年間収入
- 表面利回り

## 5. 採算性判断
- プロジェクトの実現可能性評価
- リスク要因の分析
- 推奨事項

## 6. 懸念事項
- 建設上のリスク
- 法規制上の注意点
- 地盤に関する特記事項（柱状図がある場合）
- N値分布の特異点
```
