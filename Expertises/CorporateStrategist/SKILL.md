# CorporateStrategist - 企業参謀スキル

**統合型経営支援システム**
**最終更新**: 2025-11-03
**バージョン**: 1.0

---

## 📋 このスキルについて

CorporateStrategistは、**経営者の参謀として企業経営を多角的に支援する統合型スキル**です。

4つの専門サブスキルを統合し、事業分析から人材開発、法務、洞察まで、経営に必要な機能を一元的に提供します。

---

## 🎯 4つのサブスキル

### 1. BusinessAnalyzer（事業分析）

**役割**: 事業・業務のToBe明確化

**主な機能**:
- Multiversal Structure Parser による多次元構造分析
- 事業モデルの構造化と可視化
- 業務フローの最適化提案
- 市場分析と競合分析

**詳細**: `BusinessAnalyzer/SKILL.md` を参照

---

### 2. PersonnelDeveloper（人材開発）

**役割**: 採用不可能性を前提とした人事システム

**主な機能**:
- 採用前判断（AI活用 vs 外注 vs 採用）
- 外注QCD比較による客観的評価
- 人材4類型モデル（軍人・天才・秀才・凡人）
- AIスキル化/マニュアル化による育成加速
- オペレーター → スペシャリスト への育成支援

**詳細**: `PersonnelDeveloper/SKILL.md` を参照

---

### 3. LegalAdviser（法務助言）

**役割**: 契約書作成・リーガルチェック

**主な機能**:
- 労働基準法・労働契約法等の遵守確認
- 雇用契約書・就業規則の作成支援
- 人事制度の法的妥当性の確認
- 契約書のレビューと助言

**詳細**: `LegalAdviser/SKILL.md` を参照

---

### 4. ForesightReader（洞察獲得）

**役割**: 古典的洞察に基づいた意思決定支援

**主な機能**:
- 七格剖象法による姓名判断
- 星導分析（10天体の分布とバランス）
- デジタル心易（易経による占術）
- 人材4類型判定（軍人・天才・秀才・凡人）
- 戦略的人材配置の助言

**詳細**: `ForesightReader/SKILL.md` を参照

## 🔄 使い方

### 基本的な利用フロー

CorporateStrategistは、ユーザーの要求を自動的に分析し、適切なサブスキルにルーティングします。

#### パターン1: 事業・業務の構造分析

```
ユーザー: 「事業モデルを整理したい」「業務フローを可視化したい」
    ↓
CorporateStrategist が自動判定
    ↓
BusinessAnalyzer を起動
    ↓
事業分析の実施
```

#### パターン2: 人材採用・育成・評価

```
ユーザー: 「採用すべきか判断したい」「育成プランを作りたい」
    ↓
CorporateStrategist が自動判定
    ↓
PersonnelDeveloper を起動
    ↓
人材開発の支援
```

#### パターン3: 法務確認・契約書作成

```
ユーザー: 「雇用契約書を作成したい」「就業規則を確認したい」
    ↓
CorporateStrategist が自動判定
    ↓
LegalAdviser を起動
    ↓
法務助言の提供
```

#### パターン4: 人間理解・戦略的洞察

```
ユーザー: 「この人の適性を知りたい」「人材配置を最適化したい」
    ↓
CorporateStrategist が自動判定
    ↓
ForesightReader を起動
    ↓
洞察獲得の補助
```

## 🔗 サブスキル間の連携

CorporateStrategistの強みは、4つのサブスキルが相互に連携することです。

### 連携例1: 採用判断の総合支援

```
PersonnelDeveloper: 採用前判断（外注QCD比較）
    ↓ 連携
BusinessAnalyzer: 事業戦略との整合性確認
    ↓ 連携
LegalAdviser: 雇用契約書の作成
    ↓ 連携
ForesightReader: 候補者の適性判断
```

### 連携例2: 組織再編の戦略立案

```
BusinessAnalyzer: 事業構造の分析
    ↓ 連携
PersonnelDeveloper: 人材配置の最適化
    ↓ 連携
ForesightReader: 人材の適性と相性の分析
    ↓ 連携
LegalAdviser: 組織変更の法的確認
```

## 💡 よくある質問（FAQ）

### Q1: どのサブスキルを使えばいいかわからない場合は？

**A**: CorporateStrategistに直接質問してください。自動的に適切なサブスキルにルーティングします。

例：
- 「営業事務を採用すべきか迷っている」→ PersonnelDeveloper
- 「事業構造を整理したい」→ BusinessAnalyzer
- 「雇用契約書を作成したい」→ LegalAdviser
- 「この人の適性を知りたい」→ ForesightReader

### Q2: 複数のサブスキルを同時に使いたい場合は？

**A**: CorporateStrategistが自動的に連携します。例えば、「採用判断をしたい」と言えば：
1. PersonnelDeveloper: 外注QCD比較と採用前判断
2. BusinessAnalyzer: 事業戦略との整合性
3. ForesightReader: 候補者の適性判断
を統合して提案します。

### Q3: サブスキルを直接指定できますか？

**A**: はい、可能です。

例：
- 「BusinessAnalyzerで事業分析をしてください」
- 「PersonnelDeveloperで育成プランを作成してください」

### Q4: 各サブスキルの詳細な機能を知りたい

**A**: 各サブスキルのSKILL.mdを参照してください：
- `BusinessAnalyzer/SKILL.md`
- `PersonnelDeveloper/SKILL.md`
- `LegalAdviser/SKILL.md`

## 🎯 CorporateStrategistの理念

### 基本方針

1. **統合的アプローチ**
   - 事業・人材・法務・洞察を一体的に推進
   - サブスキル間の連携による相乗効果

2. **中小企業特化**
   - 限られたリソースでの最適化
   - 実務的で即座に使えるナレッジ

3. **AI活用型経営**
   - 採用に頼らない人材マネジメント
   - AIスキル化による生産性向上

4. **人間中心設計**
   - 最終判断は人間が行う
   - AIは意思決定のサポートに徹する

## 📝 運用上の注意事項

### Weaveの立ち位置

- **支援者であり、決定者ではない**
- 客観的データと観点を提示し、最終判断は人間が行う
- 特に評価や人事判断では、評価者にはならない

### 倫理的境界線

- 業務面のアドバイザーに徹する
- 定性面（人間関係、モチベーション）には深く立ち入らない
- 法的助言は一般的な情報提供に留める（弁護士資格は不要）
